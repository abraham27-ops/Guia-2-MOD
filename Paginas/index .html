Ejemplo de MOD en PSeInt

Aplicación del operador MOD

Veamos un caso práctico del uso de MOD para determinar si un número es par o impar.

Algoritmo ParOImpar
    Definir num Como Entero
    Escribir "Ingrese un número: "
    Leer num
    Si num MOD 2 = 0 Entonces
        Escribir "El número es par."
    Sino
        Escribir "El número es impar."
    FinSi
FinAlgoritmo

Este algoritmo verifica si el residuo de dividir el número por 2 es 0. Si es así, el número es par; de lo contrario, es impar.

Otros usos del operador MOD incluyen:

  • Determinar múltiplos de un número.
  • Simular estructuras cíclicas (como semanas o relojes).
  • Alternar valores en bucles.
