Operador MOD en PSeInt

Inicio

El operador MOD en PSeInt se usa para obtener el residuo de una división entre dos números enteros.

Matemáticamente, si tenemos dos números a y b, la operación a MOD b nos devuelve el residuo de dividir a entre b.

Ejemplo: 10 MOD 3 = 1, porque 10 dividido por 3 es 3 con un residuo de 1.

Aplicación de MOD en un ejercicio

Supongamos que queremos determinar si un número ingresado es par o impar usando el operador MOD.

Algoritmo ParOImpar
    Definir num Como Entero
    Escribir "Ingrese un número: "
    Leer num
    Si num MOD 2 = 0 Entonces
        Escribir "El número es par."
    Sino
        Escribir "El número es impar."
    FinSi
FinAlgoritmo

Este algoritmo verifica si el residuo de dividir el número por 2 es 0. Si es así, el número es par; de lo contrario, es impar.
